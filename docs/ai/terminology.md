# SearchPV Terminology

## Status Terms

| User term | Internal meaning |
|---|---|
| active | Active |
| for sale | Active unless context indicates a broader search |
| en venta | Active |
| available | Usually Active |
| pending | Pending |
| under contract | Pending |
| bajo contrato | Pending |
| sold | Closed |
| closed | Closed |
| vendido | Closed |
| ventas cerradas | Closed |

## Property Type Terms

| User term | Internal value |
|---|---|
| condo | Condos |
| condominium | Condos |
| departamento | Condos |
| apartamento | Condos |
| casa | Houses |
| house | Houses |
| villa | Usually Houses; may require clarification |
| lot | Land |
| lote | Land |
| terreno | Land |

## Feature Terms

| User term | Internal interpretation |
|---|---|
| pet friendly | pets allowed according to listing data |
| accepts pets | pets allowed according to listing data |
| admite mascotas | pets allowed according to listing data |
| oceanfront | direct oceanfront classification, not merely ocean view |
| beachfront | may require beachfront or direct beach access definition |
| ocean view | primary or secondary view containing ocean |
| furnished | listing furnishing classification |
| turnkey | not assumed equivalent to furnished unless defined |
