# SearchPV Design Guide

## Goals

Simple, fast, consistent, mobile-first.

## Layout

Market → Area → Community → Development

## Cards

-   Rounded corners
-   Clean spacing
-   Consistent typography

## Navigation

-   Breadcrumbs on all hierarchy pages.
-   Consistent filters across Market, Area, Community, Development.

## Colors

Favor neutral colors. Use color only to emphasize interaction.

## UI Principles

-   Minimize clicks.
-   Let the data be the focus.
-   Keep visual hierarchy consistent.
